# PC Updater
PC UpdaterはPCをアップデートさせるつまり快適にするソフトです<br>
現在も開発中です
<br>
## PC Updaterのダウンロード方法
1.releaseから最新のPC Updaterをダウンロード<br>
2.Google Chromeだとマルウェアチェックなどが出ますが、ダウンロードを続行して大丈夫です<br>
3.ファイルを解凍し PC Updater.exeを開く

## 使い方
使用したい項目をクリックするだけです
